# Studentdoc
AI system for analyzing documents of student organizations using CrewAI and Streamlit.

# Система экспертизы документов студенческих объединений

Данный проект представляет собой систему анализа документов студенческих организаций с использованием технологий искусственного интеллекта.

## Цель проекта

Целью проекта является автоматизация процесса проверки документов студенческих объединений, выявление ошибок и формирование рекомендаций по улучшению документации.

## Основные функции системы

- анализ структуры документов
- выявление ошибок оформления
- проверка содержания документов
- формирование рекомендаций по исправлению
- создание итогового аналитического отчета

## Используемые технологии

- Python 3.12
- CrewAI
- Streamlit
- Pandas
